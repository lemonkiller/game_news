# 主题信息聚合

AI 驱动的按需信息搜索——告诉我一个主题，从所有记录的信息源搜索相关内容。

## 使用方式

直接告诉我你想看什么：

```
帮我搜 AI 辅助游戏开发
```

AI 会读取 `config/sources.yaml` 中的所有信息源，并行搜索，筛选相关内容，生成报告。

## 自定义

所有配置在项目文件中，直观可编辑：

```
config/
└── sources.yaml     # 所有信息源
template/
└── report.md        # 报告输出格式
reports/
└── *.md             # 生成的报告
```
